Greek

This documentation specifies a computational system for working with ancient Greek. The specification includes live tests verifying that an implementation fulfills the specification. The live tests are defined using concordion, so the programming language used to implement the live tests is left open.

The specification is maintained in a version control repository with implementations in JVM languages (java and groovy). A copy of the specification rendered with concordion to run all tests in this implementation is posted at http://shot.holycross.edu/~nsmith/greekspec/Greek.html

Specifications

  • Strings of characters and lexical tokens
  • Syllabification of tokens
  • Greek morphology
  • Hexameter meter
