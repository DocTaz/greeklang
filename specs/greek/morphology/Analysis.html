Morphological analyses

Every morphological analysis identifies a part of speech. Depending on the part of speech, the analysis identifies further properties. The complete set of possible properties is: person, number, tense, mood, voice, gender, case, degree.

For conjugated verbs, an analysis identifies: person, number, tense, mood and voice.

For all substantives, an analysis identifies: gender, case and number. For adjectives, it further identifies degree.

For participial forms of verbs, the "mood" is (untraditionally) considered "participle", and the analysis further identifies tense, voice, gender, case and number.

Analyses can be serialized to and created from delimited text strings giving all eight possible properties of an analysis plus a part of speech.

Example: a conjugated verb

first:singular:present:indicative:active::::verb is a complete analysis of a conjugated verb form, with the following properties set:

PropertyValue
Part of speech verb
Person first
Number singular
Tense present

Example: a noun

:singular::::masculine:genitive::noun is a complete analysis of a noun form, with the following properties set:

PropertyValue
Part of speech noun
Gender masculine
Case genitive
Number singular

Example: a participle

:singular:present:participle:active:masculine:nominative::verb is a complete analysis of a participle, with the following properties set:

PropertyValue
Part of speech verb
Mood participle
Number singular
Tense present
Voice active
Gender masculine
Case nominative
